#### 理解：

这一位的状态可以通过上一位的状态转移而来

###### 用递归写的理解：

递归出口： 其实就是初始化；

dp( i, j ) 对应 f[i] [j]

状态转移 ： f[i] [j] = f[i-1] [j]  + ...  对应  dp(i, j) =  dp(i-1, j) +...

pre变量： 当前不能转移的地方

```cpp
int f[N][N];// f[i][j]表示第i位，第i+1位数字是j的个数,
// 类比于dp计算到 f[n+1][0];
```

#### 技巧

从[l, r]满足某性质的个数  →  f(r) - f(l-1);
