#### StringBuilder类

```java

//插入
StringBuilder sb = new StringBuilder(); //对原数组操作，提升效率
str.append(s1);
str.append(34);
str.append('a');
str.toString();
//insert():指定位置插入
sb.insert(2,"qq");//在第二位后面插入qq

//删除
//StringBuffer delete(start,end):删除缓冲区中的数据，包含Start,不包含end
//StringBuffer deleteCharAt(index)；删除缓冲区指定位置的数据
sb.delete(1,3);
sb.deleteCharAt(0);

//转字符串
sb.toString()

```
